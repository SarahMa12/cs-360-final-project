Mental Health in Tech

By Sarah Ma

Project Description

This project explores the effect of workplace culture on mental health within the tech industry. Using the OSMI Mental Health in Tech Survey dataset, this interactive dashboard allows users to visualize how factors like company size, accessibility to support systems, and demographics impact employee mental health. This project will have a total of 4 visualizations, including a geographic mapping, parallel coordinates, diverging bar chart, and a sunburst diagram.
